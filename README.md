# Учебный практикум 1С: учет товаров на складе

Учебный проект на платформе **1С:Предприятие 8.3** для автоматизации базовых складских и торговых операций: номенклатура, контрагенты, поступление товаров, реализация, учет остатков, цены, продажи, управленческие проводки и отчеты.

Проект выполнен как практикум по конфигурированию и программированию в 1С. Репозиторий предназначен для демонстрации навыков junior 1С-разработчика.

## Предметная область

Система моделирует работу складского оператора: приемка товаров, хранение, учет остатков, оформление поступлений и реализаций, контроль стоимости и формирование отчетности.

Основной бизнес-контекст: складской учет и хранение товаров с использованием справочников, документов, регистров и отчетов.

## Как открыть проект

1. Создать новую пустую информационную базу 1С.
2. Открыть базу в режиме **Конфигуратор**.
3. Перейти в меню **Конфигурация → Загрузить конфигурацию из файла**.
4. Выбрать файл `configuration/УчебныйПрактикум.cf`.
5. Обновить конфигурацию базы данных.
6. Запустить режим **1С:Предприятие** и проверить работу объектов.

Подробная инструкция находится в `docs/import_instructions.md`.

## Что не публикуется

Файл выгрузки информационной базы `.dt` не добавлен в репозиторий намеренно. Он может содержать данные информационной базы, пользователей, тестовые записи и журнал событий. Для публичного портфолио безопаснее публиковать файл конфигурации `.cf` и документацию.
