# Тестовые сценарии

## 1. Проверка НСИ

1. Создать организацию.
2. Создать склад.
3. Создать единицу измерения.
4. Создать номенклатуру.
5. Создать контрагента и договор.

Ожидаемый результат: справочники заполняются, ссылки выбираются в документах.

## 2. Поступление товаров

1. Создать документ `ПриобретениеТоваров`.
2. Заполнить организацию, склад, поставщика и договор.
3. Добавить строки товаров.
4. Провести документ.

Ожидаемый результат: формируются движения прихода по остаткам и стоимости товаров.

## 3. Установка цен

1. Создать документ `УстановкаЦенНоменклатуры`.
2. Заполнить товары, вид цены, валюту и цену.
3. Провести документ.

Ожидаемый результат: цены записываются в регистр сведений `ЦеныНоменклатуры`.

## 4. Реализация товаров

1. Создать документ `РеализацияТоваров`.
2. Заполнить покупателя, договор, склад и товары.
3. Заполнить цены.
4. Провести документ.

Ожидаемый результат: остатки уменьшаются, стоимость списывается, продажи отражаются в оборотном регистре.

## 5. Отчет по остаткам

1. Сформировать отчет по остаткам товаров на складах.
2. Указать организацию и склад, если предусмотрен отбор.

Ожидаемый результат: отчет показывает остаточное количество товаров.

## 6. Отчет по продажам

1. Сформировать отчет по продажам.
2. Проверить группировки по контрагентам, документам и номенклатуре.

Ожидаемый результат: отчет выводит данные продаж из регистра накопления.
